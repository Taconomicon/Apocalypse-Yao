== README ==

* Ruby version: 2.3.0
* Rails version: 4.2.6

* Requred Packages

- Paperclip
- Imagemagick
- Devise
- SimpleForum

This is a web-forum created with Ruby on Rails. To run the server, simply clone the git, open the folder, and run: <code>rails s</code>

You may need to run the following to update the database to your local machine:
<code>rake db:migrate</code>

Additionally, you need to have Imagemagick and Paperclip installed to handle image file control.
